How to setup and start mining for I-Hash Token?

Just Click on the Mining.bat file and mining will be get started and Hashes will start stored on same folder with a file name “I-Hash.html”.
Open I-Hash.html to get the blocks information. Refresh the Browser page "I-Hash.html" after a certain time whenever you will find a new block.

If system alerts for a warning to run or download "Mining.bat", it's normal incase of Microsift windows. You can read the contents of "Mining.bat"
by right click on it and read "mining.bat" file in notepad. It must have below contents/commands:

start I-Hash.htm
java -jar IHash.jar g 1 I-Hash

How to stop or close mining application?

To Close the Mining process press CTRL+C in command prompt to get exit or close the Command prompt, but closing it properly with CTRL+C is good option.

How to claim for Your I-Hash Token? What is the procedure for it?

Once you will find a valid block hash, Click on Submit button available in I-Hash.html it will redirect you to sign a transaction via Waves wallet to Below Address:

Address: 3PG3iKrREHEC8BhRymLngyZGvcfNb7T8sNs (with an exact payment of 1 Wave)

Reason: It's a fix fees or charge to release I-Hash Tokens. (Waves transaction fees to make 1 Wave payment will be paid by miner).

Note: Tokens will not be released incase miner don't comply with above condition which is exact amount of 1 Wave along with valid hash proof.

Is it safe to sign the transaction via Waves wallet?

Yes, it safe. Becuase even after sign the transaction it will ask for confirmation to release payment. After confirmation then payment will be released.

How much time it will take to mine a block?

Finding a block hash is totally random between 4294967296 hashes from nonce, depends on luck. Time to mine a block depends on CPU Speed. Faster the CPU, faster the probability to find block.
